# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| main    | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability in FRIDAY, please report it responsibly:

1. **Do not** open a public issue
2. Email **ddeva6@gmail.com** with details
3. Include steps to reproduce if possible

You should receive a response within 48 hours. We will work with you to understand and address the issue before any public disclosure.

## Scope

FRIDAY is a research and visualization tool. It does not handle authentication, payments, or sensitive user data. However, we take all security reports seriously, including:

- Dependencies with known vulnerabilities
- Code injection risks in data processing
- Unsafe handling of external data (yfinance, NSE APIs)
